#  할리스 커피 마크업 연습

  

[할리스](https://www.hollys.co.kr/) 웹사이트를 마크업 연습

  

기존 사이트를 똑같이 마크업 하지 않고, 디자인이 좋든 나쁘든, 나만의 생각을 바탕으로

  

영역을 추가 하고 반응형 작업을 해보았다.
